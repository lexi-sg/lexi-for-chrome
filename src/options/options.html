    What data is sent, and to whom

    When you ask a question, the page's text — and, only if you use Screenshot & ask, an image of the page — are sent to Anthropic's API using your own key. No Lexi or third-party server is involved; Lexi has no backend and no account system.

    Agent Mode (letting Lexi click and type on your behalf) is off by default and must be enabled per-site. When enabled, Chrome shows its own "this extension is debugging this browser" banner for as long as it's active, and Lexi always asks before anything risky — submitting a form, navigating to a new site, paying, sending a message, uploading, downloading, or deleting.
